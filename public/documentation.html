Fellini Movies API

Fellini Movies WebApplication provided users with access to information about different Fellini movies.

Users will be able to sign up, update their personal information, and create a list of their favorite Fellini movies.

REQUEST URL HTTP METHOD
Get a list of Fellini Movies /movies GET
Get data about a single Fellini Movie /movie/[ID] GET
Add A User /user/[user name] PUT
Delete A User /user/[user name] DELETE
Update a User /user/[user name] PUT
Add a film to Favorites /movie/[ID]/[movie title] PUT
Delete a film to Favorites /movie/[ID]/[movie title] DELETE
Return Data about a Genre /movie/genre/[movie genre] GET
